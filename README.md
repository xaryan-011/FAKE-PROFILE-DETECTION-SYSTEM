Fake Profile Detection System
📌 Overview

Fake Profile Detection System is an AI-powered web application that identifies whether a social media profile is likely to be genuine or fake using machine learning. The system analyzes profile features such as follower count, following count, posts, account age, profile completeness, verification status, engagement rate, and other behavioral metrics to predict authenticity.

This project demonstrates the application of Machine Learning in cybersecurity, fraud detection, and social media safety.

🚀 Features
🔍 Detect fake and genuine social media profiles
🤖 Machine Learning-based classification
📊 Real-time prediction
📈 Confidence score for predictions
📝 User-friendly web interface
📂 Upload profile details manually or via dataset
📊 Data visualization dashboard
📉 Performance metrics 
🔒 Secure and lightweight application
